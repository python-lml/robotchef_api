Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install robotchef_api


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/robotchef_api.git
    $ cd robotchef_api
    $ python setup.py install
